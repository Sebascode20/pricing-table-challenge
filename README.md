## Overview

![screenshot](./website-screenshoot.png)

This repository contains a static HTML/CSS implementation of a pricing table UI inspired by the devChallenges design. The layout uses `flexbox` and media queries to adapt across screen sizes, and includes local font files declared in `css/fonts.css`.

## What I learned

- Building a responsive grid with `flexbox` and media query breakpoints.
- Managing theme values with CSS custom properties (variables).
- Embedding and using local fonts via `@font-face`.
- Structuring semantic HTML for a maintainable layout.

## Useful resources

- DevChallenges task: https://devchallenges.io/challenge/pricing-table-section-challenge
- MDN Web Docs (Flexbox, Media Queries): https://developer.mozilla.org/
- `@font-face` reference: https://developer.mozilla.org/en-US/docs/Web/CSS/@font-face

## Built with

- HTML5
- CSS3 (custom properties, `@font-face`, Flexbox, media queries)

## Features

- Four-plan pricing table: Free, Team, Popular (highlighted), Enterprise.
- Responsive behavior that shows/hides columns and adjusts sizes by viewport width.
- Reusable CSS variables and small component classes like `.btn-start` and `.check-container`.
- Local fonts included in `css/fonts.css`.
